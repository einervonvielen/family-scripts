# Creates a Web Pages from Images

Install

- Python
- exiftool

Run (example)

    python3 diary.py -v -i images/

What it does...

The command above searches for fotos in dir "images/" and

- reads the filename
- reads the EXIF Title (exiftool)
- reads the EXIF Description (exiftool)

Result

A file "diary.html"
